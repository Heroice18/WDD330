Chapter 5 Objects
  • I've used objects a lot for databases and Android storage. And it translates pretty well to Javascript as well.

Chapter 7 Events
  • In Web Engineering 1 we worked a lot with events as well. So this should be review again. I often use events in Android applications that have a web componet.

  • When I was developing a web interface for a project in IEEE we used keyboard events to control an external tool.
