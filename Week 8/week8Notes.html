Chapter 8 Transforms and Transitions

  • Transforms allow a property in an HTML page to be positioned properly.
  • Translation allows an element to be moved in a 4 way direction.

  • Scaling is useful when you want to size up or down an element when put in a different resolution size.

  • Rotation can rotate an element around a specific point.

  • Animations can be useful as they allow the site to feel more dynamic and fluid.

Chapter 12 Canvas, SVG, and Drag and Drop
  • Canvas allows us to draw any shape with any coloring and add it to the site through Javascript.
  • When we have a canvas element we want to use, we also have to create a context to identify where we place the canvas.

  • We can also create images using SVG (Scalable Vector Graphics)

  • SVG's allows the quality of the image to be preserved even upon growing or shrinking.
