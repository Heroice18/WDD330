Validating forms

  • I've used a lot of forms and feedback in my sites when I was a frontend engineer.
  • Having these sets of rules makes it very easy to error check so that the backend doesn't recieve invalid data.

  • By setting specific boundaries we can also limit and control what the user inputs into the field.

Using Fetch
  • We can use fetch to retrieve data over the network such as json or other forms of data to then manipulate onto our webpage.
  • We can retrieve or upload data to the server using fetch

  • Using this tool you can have data being sent and retrieved while other aspects of the program are running.
